Emmit Sublime Package

Emmet is a helpful Sublime Text package that makes writing html code more efficient. To begin type the element you would like to use. Divs do not need to be specified if they are at the beginning of the line. For instane to create a paragraph type p + tab. To add something after an element use the + sign. To add two paragraphs type p + p. To insert things into an element use {}. Here is how to do a paragraph with text, p{text}. Putting a . or # after an element allows you to add an id or class attribute. p.hello creates a paragraph with the class hello. To nest an element inside of another element use the > charachter. ol>li{item 1}*6. The code previously inserts 6 list items into an ordered list. The 1 symbol tells emmet to insert the number of the item in the list. Using *, you can tell emmet to make x numbers of an element. Using emmet you can write out HTML with ease. Once you have written what you want emmet to create, hit tab to complete the code. You must also be at the end of the line for completion to work.

Make a Header

To make a header using emmet. Type h1.item_header{Header} + tab. This creates a h1 element with a class item_header, and the content "Header".
